                            //reverse the string
                            // logic function
                            function reverseString(userString) {
                              let revString = [];
                              for (let index = userString.length - 1; index >= 0; index--) {
                                revString += userString[index];
                              }
                              return revString;
                            }
                            
                            // display success message with reversed string
                            // view function
                            function showString(revString) {
                              document.getElementById(
                                "message"

                              document.getElementById("alert").classList.remove("invisible");

The code is structured in three functions

getValues

The getValues function begins with using the document method getElementById to find the element with the "alert" id and add the class of "invisible" to it. The variable userString calls on the document method getElementById to find the element with the id of "userString" from the input element. The value property is appended to document method getElementById to get the value contained within the input element.

reverseString

The reverseString function takes one argument, userString. The function reverseString is called in the getValue funtion after userString is set. The variable revString is set to an empty array. In our for loop the index is set to the length of the userString - 1 to begin the string at the end of the index. As long as the index is great than or equal to zero, the final call of index-- is a call to decrement the count. The we incramentally add the the index of the userString. Finally we return revString

showString

Finally, the showString which takes the argument revString is called. The showString funtion uses the document method getElementById to find the element with the id of "message". The propert innerHTML is appendend to the element with the id of message. The innerHTML is set to a message that that informs you what the reversed string is. To ensure that the string is dynamic, a template literal is used to call revString. Finally the document methond is used again to find the element with the id of alert, this time the class of invisible is removed from the element.
